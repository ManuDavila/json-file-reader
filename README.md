json-file-reader
================

<div>
<h1>php json file reader:</h1>
<h5>This class allow open and read a json file and extract the content it in array format</h5>
<h3>Use: </h3>
<pre>
$reader = new json_file_reader();
$json = $reader->json("images.json");
print_r($json);
</pre>
